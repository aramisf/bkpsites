Aula 12

Filtragem usando grep

- uso do grep em linhas de comando simples
- opções do grep (-n, -c, -l) 
- filtragem em múltiplos arquivos e diretórios
- filtragem em arquivos compactados
- possibilidade de uso do grep para buscas rápidas (opção -m)
- pesquisa por padrão e contexto (opção -C)

Exercícios:

1) Procure a partir do diretório /usr no sistema
todos os arquivos em fomato ASCII text.

2) Encontre no diretório /usr/share/doc/HOWTO/en-txt
todos os arquivos que possuem a palavra clustering.

3) Construa uma linha de comando usando ls e grep
para mostrar apenas os arquivos e somente os
arquivos.

4) Escreva um comando para contar o número de linhas
do arquivo /etc/passwd que possuem a palavra bash.

5) Escreva um comando para encontar todos os usuários
Que possuem Souza no seu nome. Mostre apenas o login
e o nome completo.
