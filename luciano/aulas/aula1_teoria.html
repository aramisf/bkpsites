Conta no sistema
- Login (usuário) e password (senha): são ambos case-sensitive
- Diretório padrão (HOME) do usuário: /home/bcc/fulano onde fulano=login
- Senha deve ter 9 dígitos, não deve ser uma palavra de qualquer
  dicionário ou idioma, deve conter letras e números embaralhados, não 
  deve ser sua data de nascimento, parte do seu RG, RA, etc. 
  Deve ser como: 2Wj93$qP
  Uma dica é usar uma frase simples para compor sua senha
  A senha tem 6 Letras e 2 Numeros -> Ast6Le2N
- Problemas com sua conta informe imediatamente o funcionário do Lab.
- Leia com atenção as normas de uso do laboratório:
Normas

Como usar o sistema
- Escolha um servidor de processamento
- Digite login e password (case-sensitive)
- Como funciona o ambiente gráfico
- Navegue pelo menu de aplicações
- Navegador web (firefox, galeon, opera)
- Editor de texto (gedit, emacs, vi)
- Encontre o programa xterm na barra de ferramentas do gerenciador de janelas

Como executar comandos
- Terminal (xterm) e SHELL
- Estrutura da linha de comando:
  <nome>   <opcoes>   <argumentos>
  Ex: ls -l /home
- Comandos podem ou não precisar de opções e argumentos.
- Como encontrar o manual on-line dos comandos (man)
- Alguns comandos básicos: date, cal, ls
- Para trocar a senha usa-se o comando passwd

Como enviar e receber email
- Use o seu email externo
- Pode-se redirecionar os emails do DInf para seu email externo
- WebMail do Dinf em: http://www.inf.ufpr.br/mail
- As mensagens ficam o arquivo Mailbox da sua área
