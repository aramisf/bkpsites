Exercícios - Aula 23


1. Explique o script abaixo:

#!/bin/bash

cal

2. Escreva um script para mostrar o PID de todos os processos do seu usuário.

3. Escreva um script baseado no exercício 2 para matar todos os processos do seu usuário.

5. Faça um script para guardar backup da sua área em um arquivo cujo nome deve conter a data de criação do backup. Ex: backup_luciano_10Nov2004.tgz

6. Anote suas dúvidas para a próxima aula.
