Exercícios - Aula 15


1. Cite 3 exemplos práticos para o uso do comando awk.

2. Explique o que faz cada parte do comando abaixo:
$ ls -l | grep -v ^to| awk '{print "perm="$1" tamanho="$5}'

3. Crie algumas linhas de comandos usando variáveis pré-definidas do awk. Para isso leia a seção 7 Builtin-variables no man do awk.

4. Usando a função toupper(s) do awk mostre o nome dos usuários logados no sistema somente com letras maiúsculas. Para isso você pode usar a saida do comando who como entrada para o awk.

5. Usando a tabela de notas da disciplina (faça um copy/paste com o mouse na página, insira em um arquivo e atribua uma nota para a segunda prova), calcule a média parcial da primeira com segunda prova (veja os pesos de cada prova) para cada aluno e imprima a media seguido do nome do aluno. Para isso use o comando awk. DICA: '{print ($3+$4*2)/3FS$1FS$2}'

6. Usando a mesma tabela do exercício anterior, mostre qual a nota que o aluno deve tirar na terceira prova (que tem peso 4) desta disciplina, para que sua média final seja no mínimo 7. Use o comando awk para isso.

7. Anote suas dúvidas para a próxima aula.
