Laboratório 13

Programação shell, sed(1) e expr(1)

Crie scripts shell que sejam solução para os exercícios abaixo. ATENÇÃO: cada script deve ser executado como se fosse um comando normal do UNIX.

  1. Na frase O gato preto foi caçado pelo cachorro marrom, substitua a palavra preto por branco.

  2. O script recebe 2 (dois) argumentos: um arquivo e uma expressão regular. Usando apenas sed(1), o script deve mostrar APENAS as linhas do arquivo que contém o padrão representado pela expressão regular.

    DICA: veja a opção -n em sed(1).

  3. Dado um arquivo como argumento do script, existem 3 maneiras de mostrar apenas as 10 (dez) primeiras linhas usando sed(1). Ache estas 3 soluções.

  4. Dado um arquivo como argumento do script, remova todas as linhas que tenham a palavra debug.

  5. Dado um arquivo como argumento do script, remova todas as linhas que tenham a palavra debug e mostre apenas as linhas que contenham a palavra teste (mas que, evidentemente, não contenham debug).

    Este script pode ser resolvido usando-se apenas grep(1) ou, alternativamente, usando-se apenas sed(1) apenas uma vez. Encontre estas duas soluções. Em seguida responda: neste caso, qual a vantagem em se usar a solução com sed(1)?

  6. Usando sed(1), em combinação ou não com outros comandos UNIX, conte todos os caracteres ALFABÉTICOS de um string passado como argumento.

  7. Usando expr(1), conte todos os caracteres de um string passado como argumento.
