CI066 - Oficina de Programação

Ementa: Uso dirigido de ferramentas de programação. O computador como ferramenta de trabalho. Práticas de programação.

Objetivos: Apresentar o ambiente de trabalho como um conjunto de ferramentas de programação e dar ao aluno condições de resolver problemas relativamente complexos através no uso destas ferramentas.

FAQs:

  • Como acessar o webmail do DInf?
    A URL é a seguinte: http://www.inf.ufpr.br/mail
  • Existe alguma forma de acesso remoto para que eu possa fazer os exercícios nos quais preciso ver a árvore de diretórios do laboratório?
    Sim. execute "ssh dupond.inf.ufpr.br" ou "ssh dupont.inf.ufpr.br", se você está em uma máquina Linux.
    Se está em um sistema MS-Windows, leia o texto em http://www.inf.ufpr.br/info/depto/parq_comp.html
