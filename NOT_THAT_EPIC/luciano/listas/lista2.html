Exercícios - Aula 2


1. Acesse o sistema com sua conta e veja como você faz para sair dele. Verifique a opção Ctrl+Alt+Backspapce.

2. Abra um terminal (xterm), escolha uma boa senha e use o comando passwd para trocá-la (se voce não fez na aula1).

3. Navegue pelo diretório /usr/share/doc/HOWTO e acesse alguns arquivos com o comando zless.

4. Navegue pelo sistema de arquivos, use os comandos file, less, man para saber o que representa cada um dos arquivos, principalmente os comandos dos diretórios /usb/bin e /bin.

5. Execute o comando (aplicativo) firefox, que é o navegador web recomendado para uso no Dinf e acesse a documentação do sistema via file:///.

6. Teste algumas opções do comando ls em seu diretório HOME.

7. Acesse outros aplicativos disponíveis no seu gerenciador de janelas para se familiarizar com o sistema.

8. Acesse o sistema do DInf de uma máquina externa via ssh, ou winssh.

9. Anote suas dúvidas para a próxima aula.
