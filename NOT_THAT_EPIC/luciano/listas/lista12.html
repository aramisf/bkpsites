Exercícios - Aula 12


1. Escreva uma expressão regular para encontrar todas as palavras que começam com "te" e terminam com "a", que possuem o caracter "o" e que estão no ínicio da linha em um arquivo.

2. Escreva uma linha de comando para encontrar todos os usuários no sistema que possuem "silva" no seu nome. A saída deve ser filtrada para aparecer o login do usuário seguido do seu nome completo.

3. Escreva outra linha de comando para contar quantos usuários foram encontrados no exercício anterior. A linha de comando deve gerar um arquivo com os nomes ordenados alfabeticamente.

4. Escreva uma expressão regular para encontrar em arquivos as linhas que contenham as palavras casa, casas ou casadas.

5. Explique a linha de comando abaixo: 

     find /usr/ -type f -exec sh -c "file {} | grep -q ASCII && ls -l {}" \;

6. Anote suas dúvidas para a próxima aula.
