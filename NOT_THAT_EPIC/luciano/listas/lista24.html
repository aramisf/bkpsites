Exercícios - Aula 24


1. Escreva um script que testa quantos parâmetros foram passados na linha de comando. Se o número for diferente do que é necessário para o script, então imprima com se deve usar o comando. DICA: use o comando exit para parar a execução do script.

2. Leia a seção CONDITIONAL EXPRESSIONS no manual do bash.

3. Escreva um script que recebe 3 valores inteiros positivos na linha de comando e imprime o maior deles. Resolva este exercício usando o menor número de if e else.

4. Resolva o exercício anterior, porém encontre o menor valor.

5. Resolva o exercício anterior, porém encontre o valor do meio.

6. No início dos scripts dos exercícios 3, 4 e 5, insira algumas linhas de código para verificar se os valores digitados como parâmetros são realmente valores inteiros positivos. Dica: use uma expressão regular para representar os inteiros positivos.

7. Anote suas dúvidas para a próxima aula.
