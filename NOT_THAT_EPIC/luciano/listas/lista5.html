Exercícios - Aula 5


1. Verifique o que faz os comandos Ctrl-g, Ctrl-s e Ctrl-r no emacs.

2. Execute comandos, como o ls por exemplo, dentro do vi e do emacs.

3. Tente editar um arquivo texto com o emacs usando apenas teclas de atalho, evitando fazer uso do mouse. Caso você não saiba quais são as teclas de atalho para determinada ação procure no manual on-line, ou guia de referencia, ou documentação pública sobre o emacs.

4. O que faz o comando ESC+% no emacs.

5. Como você faz para copiar uma linha de um arquivo no vi e inserí-la no final do arquivo.

6. Resolva o exercício 5, mas para 5 linhas consecutivas.

7. Resolva os exercícios 5 e 6 usando o emacs.

8. Qual o comando para remover uma linha inteira no vi e no emacs.

9. Verifique o funcionamento do ESC+l e ESC+u sobre uma palavra no emacs.

10. Execute set -o emacs e veja o que fazem as teclas de atalho do emacs no bash. Uma dica interessante é o Ctrl-r no bash, o que ele faz?

11. Resolva o exercício 10 para o vi usando o set -o vi.

12. Explique como funciona as permissões em arquivos e diretórios.

13. Escreva um comando para trocar todas as permissões de arquivos terminados com .txt do diretório /tmp. As permissoes devem ser apenas para leitura e escrita para voce e sem permissão para mais ninguém.

15. Anote suas dúvidas para a próxima aula.
