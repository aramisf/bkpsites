Revisão 3


1. Faça todos os exercícios das aulas anteriores.

2. Leia a bibliografia recomendada para esta disciplina.

3. Escreva scripts para todos os exercícios anteriores da disciplina, tornando-os genéricos, ou seja, que parâmetros sejam passados na linha de comando.

4. Escreva um programa para que dada uma data (dia mes ano) diga o dia da semana. DICA: veja a saída do comando cal 10 2050 que mostra o dia da semana para todos os dias do mês 10 do ano de 2050.

5. Escreva um programa para encontrar feriados no meio da semana nos próximos 5 anos, como por exemplo dia 07 de setembro, 15 de novembro, etc. Crie uma lista destes feriados em um arquivo de entrada para o seu programa.

6. Escreva um programa para encontrar os dois usuário no sistema que possuiem respectivamente o maior e o menor nome, considerando nome e sobrenomes. Se houver empate use a ordenação por letras (veja no man do bash string1 < string2). DICA: os usuários estão no /home/bcc

7. Escreva um programa que inicialmente lê dois vetores A e B de números inteiros positivos. O programa deve criar um terceiro vetor C que contém os elementos do vetor A e B ordenados. Ao final deve imprimir os vetores.

8. Escreva um script para gerar automaticamente um arquivo com 50 linhas e 4 colunas, separadas por ":" contendo na primeira coluna um login de até 8 digitos (6 letras seguidas de dois numeros equivalente ao ano de entrada, neste caso considere apenas do ano de 2000 a 2006), e nas colunas seguintes um valor inteiro entre 0 a 10. Ou seja, o arquivo possui um login e 3 notas. Cada linha deve ser gerada randomicamente utilizando a variável RANDOM do sistema. DICA: veja como imprimir com o comando echo as letras em ASCII utilizando octal (echo -e "\0101" # gera um A)

9. Escreva um script para ordenar um arquivo alfabeticamente, sendo que este arquivo é separado por colunas (com separador ":"). O programa deve receber 1 ou 2 parâmetros na linha de comando. O primeiro é o nome do arquivo a ser ordenado e o segundo é opcional, que seria o arquivo de saída. Havendo apenas 1 parametro o programa deve sobrescrever o arquivo de entrada, caso contrário, havendo 2, o programa preserva o arquivo original e salva o resultado no arquivo passado como 2o. parâmetro. Lembre-se de checar os possíveis erros na linha de comando.

10. Altere o script 9 para que o mesmo possa receber uma opção -c, que identifica a coluna utilizada para ordenação. Ou seja, se o usuário passar o parâmetro -c3, o script ordena pela coluna 3. A ordem dos parametros é primeiro as opções depois os argumentos.
