CI066 - Oficina de Programação (aulas)


Ementa: Uso dirigido de ferramentas de programação. O computador como ferramenta de trabalho. Práticas de programação.

Objetivo: Apresentar o ambiente de trabalho como um conjunto de ferramentas de programação e dar ao aluno condições de resolver problemas relativamente complexos através do uso destas ferramentas.

Sistema de avaliação:
    Notas das provas são atribuídas de 0 a 10.
    Média = (Prova1+2*Prova2+4*Prova3)/7
    Reprovado por falta se Freqüência < 75%
    Reprovado por nota se Média < 4.0
    Em Exame Final se 4.0 <= Média < 7.0. NotaFinal = (Média + ProvaFinal)/2
    Aprovado se Média >= 7.0 ou NotaFinal >= 5.0
